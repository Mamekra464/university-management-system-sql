# University Management System (SQL Database Project)

## Project Overview
This project involved designing and implementing a relational database for managing university operations, including students, courses, faculty, departments, enrollments, and library records.

The database was built using SQL and demonstrates skills in database design, normalization, query development, and reporting.

## Key Features
- Relational database design with multiple entities
- Primary and foreign key relationships
- Database normalization up to Third Normal Form (3NF)
- CRUD operations (Create, Read, Update, Delete)
- Complex SQL queries using JOIN, GROUP BY, and aggregate functions
- Views and stored procedures
- Indexing and query optimization
- Reporting and analytics using SQL

## Database Entities
- Students
- Courses
- Faculty
- Departments
- Enrollments
- Library

## Skills Demonstrated
- SQL programming
- Database design and normalization
- Data analysis and reporting
- Query optimization
- Data integrity and constraints
- Analytical thinking

## Files Included
- SQL script for database creation and queries
- Project documentation with query outputs and reports
